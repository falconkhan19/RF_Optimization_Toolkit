# RF Optimization Toolkit

A professional QGIS plugin for LTE and 5G RF Planning & Optimization.

## Features

- LTE KPI Import
- 5G NR KPI Import
- Automatic KPI Join
- SQLite Database
- Sector Generation
- KPI Visualization
- Neighbor Analysis
- PCI Analysis
- Drive Test Analysis
- Reporting

## Requirements

- QGIS 3.40+
- Python 3.x
